1. main program name : case86BMD_DK.m
2. Input data:  first column is dose, second column is response, third column is case number.
3. There are four parts of the main program:
         First part, get the dataset of each case;
         Second part,  fit the Kriging models based on sampling data;
         Third part, get the BMD of BMR = 0.1;
         Fourth part, get the BMDL through Bootstrapping algorithm.
